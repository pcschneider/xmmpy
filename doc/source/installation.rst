Installation
============================

Required packages
-----------------
xmmpy requires:
  - numpy
  - astropy

Paths
-----
The following paths need to be set::
 
  export xmmpy=/home/majestix/hdd/tools/xmmpy
  export PATH=$PATH:${xmmpy}/bin
  export PYTHONPATH=$PYTHONPATH:/home/majestix/hdd/tools
